
# Software Engineering 2026

Welcome to my Software Engineering coursework repository. This project serves as a central hub for all lab exercises, classworks, and assignments completed throughout the semester.

## 📌 Project Overview

This repository tracks my progress in software development, covering various programming languages and engineering principles.

* **Primary Languages:** JavaScript (71%), C# (18.3%), HTML/CSS, and ASP.NET.
* **Key Focus:** Application logic, UI/UX design, and database integration (TSQL).

---

## 📁 Repository Structure

The project is organized by laboratory sessions and specific assignments:

| Directory/File        | Description                                                       |
| :-------------------- | :---------------------------------------------------------------- |
| `.github/workflows` | CI/CD configurations for cloning and running the repository.      |
| `Lab1` to `Lab6`  | Source code and assets for weekly lab exercises.                  |
| `QT2Assignment1`    | Assignment 1 for Quarter 2, including data visualizations/charts. |
