## 📌 Project Overview

The **Ticket Vendor Machine (TVM)** system is designed for the Ben Thanh Metro line in Ho Chi Minh City. The system facilitates the purchase of bus and Metro (MRT) tickets, supporting modern payment methods including **Credit Cards** and **QR Code** payments (VietQR, MoMo, VNPay, ZaloPay).

## 🛠 Tech Stack

* **Backend:** C# (.NET Core 8.0/9.0)
* **Architecture:** MVC (Model-View-Controller)
* **Database:** Microsoft SQL Server (MSSQL) with EF Core (Code-First)
* **Design Tools:** Draw.io, Figma, Canva

---

## 📂 Project Structure

Based on the repository layout:

* `AI Logs/`: Transcripts and compact summaries of AI pair-programming sessions (`claude.txt`, `gemini.txt`).
* `Charts/`: System architecture and design diagrams (`charts.drawio`, `usecase.png`, `activitydiagrams.png`, etc.).
* `HCMCMetroKioskV1/`: The core .NET Core MVC application directory containing the `.sln` file, `Controllers/`, `Models/`, `Views/`, and `wwwroot/`.
* `Requirements.xlsx` & `usecaseDesc.xlsx`: Detailed system requirements and use case specifications.
* `Notes.md` & `readme.md`: Internal developer notes and main project documentation.
